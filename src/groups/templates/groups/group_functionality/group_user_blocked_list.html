About Blocking
What blocking does:
  • Prevents the user from joining the group
  • Blocks all join requests from the user
  • Removes them if they're currently a member
  • They cannot see group details
After unblocking:
  • User can send join requests again
  • They are NOT automatically added back
  • You'll need to approve their request
  • No notification is sent to them
