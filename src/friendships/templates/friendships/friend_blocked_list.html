About Blocking
What blocking does:
  • Prevents friend requests from this user
  • Hides your detailed profile from them
  • Removes existing friendship if any
  • Blocks all direct interactions
After unblocking:
  • User can send friend requests again
  • Previous friendship is not restored
  • You'll need to reconnect if desired
  • No notification is sent to them
